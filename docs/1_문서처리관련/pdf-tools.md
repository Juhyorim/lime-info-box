# PyMuPDF vs pdfplumber vs Tesseract OCR 비교 정리

---

## 1. 기본 개요

| 도구               | 기반                     | 처리 방식                      | 특징                          |
| ------------------ | ------------------------ | ------------------------------ | ----------------------------- |
| **PyMuPDF (fitz)** | MuPDF (C 라이브러리)     | PDF 내 텍스트 레이어 직접 추출 | 범용 PDF 처리, 속도 매우 빠름 |
| **pdfplumber**     | pdfminer.six             | PDF 내 텍스트 레이어 직접 추출 | 표(Table) 추출에 특화         |
| **Tesseract OCR**  | Google 오픈소스 OCR 엔진 | 이미지에서 문자 패턴 인식      | 스캔 PDF / 이미지 PDF 전용    |

---

## 2. 기능별 비교

| 항목            | PyMuPDF       | pdfplumber      | Tesseract       |
| --------------- | ------------- | --------------- | --------------- |
| 디지털 PDF 지원 | ✅            | ✅              | ❌              |
| 스캔 PDF 지원   | ❌            | ❌              | ✅ (핵심 기능)  |
| OCR 기능        | ❌            | ❌              | ✅              |
| 표 추출         | 보통          | **우수**        | 미흡            |
| 이미지 추출     | ✅ (강력)     | ❌              | -               |
| PDF 편집/생성   | ✅            | ❌ (읽기 전용)  | -               |
| 처리 속도       | **매우 빠름** | 상대적으로 느림 | 느림            |
| 한국어 지원     | 텍스트 OK     | 텍스트 OK       | kor 언어팩 필요 |

---

## 3. 각 도구 상세

### PyMuPDF

**주요 용도**

- 빠른 텍스트 추출
- 이미지 추출 및 PDF → 이미지 변환
- 메타데이터 읽기
- PDF 생성·수정·병합

**장점**

- 속도가 가장 빠름
- 범용적 (텍스트, 이미지, 편집 모두 가능)
- 좌표/폰트/크기 등 레이아웃 정보 제공

**단점**

- 표 구조 파악 약함 (직접 파싱 로직 필요)
- 스캔 PDF 불가 (OCR 별도 연동 필요)

---

### pdfplumber

**주요 용도**

- 정형화된 표가 많은 문서 처리
- 레이아웃 세밀 분석
- 좌표 기반 특정 영역 텍스트 추출

**장점**

- 표 추출 압도적으로 강함 (선 기반 감지)
- `page.extract_table()` 한 줄로 표 추출 가능
- 텍스트 위치 정보 세밀하게 분석

**단점**

- 속도 느림
- 이미지 처리 불가
- 읽기 전용 (편집 불가)
- 스캔 PDF 불가

---

### Tesseract OCR

**주요 용도**

- 스캔 문서에서 텍스트 추출
- 이미지 PDF 처리
- 손글씨 인식 (제한적)

**장점**

- 스캔 PDF 유일한 해결책
- 한국어 지원 (kor+eng 언어팩)
- 오픈소스 무료

**단점**

- 처리 속도 가장 느림
- 표 구조 인식 약함
- 이미지 품질에 따라 정확도 차이 큼
- 디지털 PDF에는 사용 불필요

---

## 4. 언제 무엇을 쓸까?

| 상황                        | 추천 도구     |
| --------------------------- | ------------- |
| 일반 PDF 빠르게 텍스트 추출 | PyMuPDF       |
| 표가 많은 PDF 정확하게 처리 | pdfplumber    |
| 이미지/스캔 PDF 텍스트 추출 | Tesseract OCR |
| 이미지 추출 or PDF 편집     | PyMuPDF       |
| 복잡한 레이아웃 분석        | pdfplumber    |

---

## 5. RAG 프로젝트 추천 처리 순서

```
1단계: PyMuPDF로 텍스트 추출 시도
    ├─ 텍스트 정상 추출  →  완료 ✅
    ├─ 표 구조 감지      →  pdfplumber로 전환
    └─ 텍스트 없음       →  Tesseract OCR 적용
```

> **TIP:** 한/영 혼용 스캔 문서라면 Tesseract 언어 설정을 `kor+eng`로 지정할 것
